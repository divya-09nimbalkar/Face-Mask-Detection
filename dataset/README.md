# Dataset for Face Mask Detection

This folder contains the training data used by `face_mask_detector.py`.

Place your face images into the following subfolders:

- `with_mask/` : images of people wearing masks
- `without_mask/` : images of people without masks

Supported image formats:
- `.jpg`
- `.jpeg`
- `.png`
- `.bmp`

The script ignores all other file types.
